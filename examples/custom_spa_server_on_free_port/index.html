Example: Web-Server that finds available free port

Similar to the Custom Web-Server example, this HTML page is handled by a custom Web-Server other than WebUI.
This window is connected to the back-end because we used: 

http://localhost:[WEBUI_PORT]/webui.js
An available free port ([WEBUI_PORT]) was obtained by calling 
webui.getFreePort();
